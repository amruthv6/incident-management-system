# Incident Management System

## Overview

Incident Management System is an operational tracking platform designed to manage incidents, monitor issue resolution workflows, and maintain structured records of operational events.

The application enables users to create incidents, assign priorities, track resolution status, and monitor operational performance through centralized visibility.

This project simulates operational environments used in IT support, infrastructure monitoring, and workflow escalation systems.

---

## Key Features

### Incident Tracking
- Create incidents
- Update incident status
- Track issue lifecycle

### Priority Classification
- Low
- Medium
- High
- Critical

### Incident Workflow
- Open
- Investigating
- Resolved

### Dashboard Metrics
- Active incidents
- Resolved incidents
- Critical incidents
- Resolution trends

### Search & Filtering
- Filter by:
  - Priority
  - Status
  - Incident type
